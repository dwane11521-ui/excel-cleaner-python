# Excel Cleaner Python

A simple Python tool for cleaning Excel files.

This script reads an Excel file, removes empty rows and duplicate rows, then saves a cleaned Excel file.

## Features

- Read Excel files
- Remove completely empty rows
- Remove duplicate rows
- Export a cleaned Excel file
- Simple and beginner-friendly Python script

## Requirements

Install the required packages:

```bash
pip install -r requirements.txt
```

## How to Use

1. Put your Excel file in the project folder.
2. Rename the file to:

```text
input.xlsx
```

3. Run the script:

```bash
python main.py
```

4. The cleaned file will be saved as:

```text
cleaned_output.xlsx
```

## Example

Input file:

```text
input.xlsx
```

Output file:

```text
cleaned_output.xlsx
```
